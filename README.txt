SOAP

then go to the url 
http://localhost:8080/SOAPClient/sampleFilmDAOProxy/TestClient.jsp?endpoint=http://localhost:11668/SOAP/services/FilmDAO

REST
run the com.restapi.ryan.film on a tomcat 9 server

http://localhost:8080/com.restapi.ryan.film/rest/films
and for html file 
http://localhost:8080/com.restapi.ryan.film/create_film.html
